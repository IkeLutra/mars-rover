# Mars Rover Test

## Requirements

- Rust 1.20 ([Install instructions](https://www.rust-lang.org/tools/install))

## Running

With Rust installed to run the CLI:

```
cargo run <path/to/input_file>
```

To run tests
```
cargo test
```
